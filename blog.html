VAR

var declarations are globally scoped or function scoped. The scope is global when a var variable is declared outside a function. var is function scoped when it is declared within a function. var variables can be updated and re-declared within its scope.

LET

let declarations are block scoped. let variables can be updated but not re-declared.

CONST

const declarations can only be accessed within the block they were declared. const variables can neither be updated nor re-declared.

NULL

Null is an assignable value, and it is sometimes intensionally assigned to a variable to show the absence of the value of that variable. When we need to indicate no assigned value for a variable or object, we use null.

UNDEFINED

It indicates variables that are declared but not assigned any values or something that is "not defined." In javascript, when we declare a variable and do not assign it any value, it is called an undefined variable. It is automatically assigned by Javascript to the variable if we forget to assign any value to a variable.

REST API

REST (Representational State Transfer) API is an architectural style for designing networked applications. In other words we can say, APIs that follows REST architectural style are known as REST API.
